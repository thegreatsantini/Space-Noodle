# Space Noodle

Play the game [here](http://spacenoodle.surge.sh/)

## to contribute

First fork repo

Clone 
``https://github.com/thegreatsantini/Space-Noodle.git``

IInstall dependencies
``npm install``

Run local sever
``npm run start``
